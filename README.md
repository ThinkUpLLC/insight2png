# Requirements:

* [slimerjs](http://slimerjs.org/)
* xvfb (`apt-get install xvfb`)

# Run:
```
xvfb-run slimerjs server.coffee
```

Point browser to:

```
http://localhost:8080/insight.png?url=https://ginatrapani.thinkup.com/?u=ginatrapani&n=twitter&d=2014-08-28&s=weekly_best
```

And get an image!
